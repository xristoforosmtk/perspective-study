# Perspective Study: 3-Point Perspective Simulator

An interactive Python-based tool designed to visualize and study **3-point perspective** using projective geometry principles. Unlike simple linear scaling, this project implements non-linear compression to accurately simulate how objects diminish toward vanishing points.

## Key Features
* **Interactive UI:** Dynamic sliders for real-time adjustment of Vanishing Points (VPs), Horizon levels, and object dimensions.
* **Projective Interpolation:** Custom algorithm using homogeneous coordinates to create a realistic "Projective Ruler" grid.
* **Dynamic Rendering:** Supports both wireframe and solid-face visualization with depth-aware face ordering.
* **Mathematical Precision:** Uses geometric cross-ratios to ensure visual accuracy in spatial compression.

## Tech Stack
* **Language:** Python
* **Libraries:** Matplotlib, NumPy, Ipywidgets
* **Environment:** Best viewed in Jupyter Notebook or VS Code Interactive window.
